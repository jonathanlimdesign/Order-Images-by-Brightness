#Order Images by Brightness

The Order Images by Brightness is a tool that renames and reorders images in a folder by their brightness value to allow a better viewing experience.

##Usage

1. Open the "orderByBright.py" file in a code editor

2. Run the code

3. Input the absolute file path when prompted with "What is the file location for organization: "

4. Input the desired base file name when prompted with "What is the name of this material?: "

5. The specified folder should have all its images reordered by brightness.
